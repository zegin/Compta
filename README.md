# Compta

## Instal and start

### Start Server
  ```bash
    cd server
    npm i
    nodemon index.js
  ```

### Start App
  ```bash
    cd ..
    npm i
    npm run dev
  ```

## Contribution
  * Test your code and **lint**
      ```bash
        npm run lint
      ```
  * If necessary change version
      ```bash
        git tag -a [version]
      ```
  * Push
      ```bash
        Push origin dev --tags
      ```
